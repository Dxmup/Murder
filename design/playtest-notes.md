# Playtest and Simulation Index

The current design has completed four isolated-agent baseline runs. Full outputs remain under [`../simulation/runs/`](../simulation/runs/); they are test records, not alternate game content.

| Run | Finding |
|---|---|
| Baseline 01 | Progressive dominated because the sequence was packaged too explicitly. |
| Baseline 02 | Removing answer-shaped language reduced but did not eliminate Progressive dominance. |
| Baseline 03 | Stricter evidence expectations overcorrected toward Human Mantle. |
| Baseline 04 | Synthetic Origin and Human Mantle split 8–8; One Human and Progressive received no votes. |

Detailed findings: [`../simulation/baseline-01-findings.md`](../simulation/baseline-01-findings.md), [`baseline-02`](../simulation/baseline-02-findings.md), [`baseline-03`](../simulation/baseline-03-findings.md), and [`baseline-04`](../simulation/baseline-04-findings.md).
