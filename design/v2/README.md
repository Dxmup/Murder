# Version 2 — Rebuild

Version 2 is a clean-sheet reconstruction of *How Many R's in Murder?* The existing roster, sheets, messages, and simulations remain useful historical material, but they are not constraints on this version.

Start with [`core-concept.md`](core-concept.md), then [`cast-and-mechanics.md`](cast-and-mechanics.md) and [`messaging.md`](messaging.md). The cast and provisional relationship graph now also exist as machine-readable tables in [`../data/characters.csv`](../data/characters.csv) and [`../data/relationships.csv`](../data/relationships.csv).

The next design artifact is the equal-weight ending/evidence matrix, followed by the detailed Vale and article-rumor timelines. Only after those are stable should the project replace the legacy facts, events, evidence, and messages CSVs or write full V2 booklets.
